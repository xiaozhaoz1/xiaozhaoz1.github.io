项目简介

LittleMaidMoreAction 是车万女仆（Touhou Little Maid）的附属模组，为女仆添加可视化规则编辑器与事件驱动的战斗动作系统。基于 Minecraft 1.20.1 Forge，Java 17，169 个源文件。

GitHub: xiaozhaoz1/LittleMaidMoreAction

核心功能

可视化规则编辑器

不用手写 JSON，通过游戏内 GUI 即可配置复杂的战斗规则。三条预设（处决/闪避/弹反）开箱即用，27 种事件、54 种动作、35 个条件键全覆盖。

架构设计

层 包 职责

适配层 adapter/tlm/ TLM API 隔离，事件桥接，MagicCasting Provider

开发历程

从 v3 的 JSON 驱动到 v5 的注解驱动加 SPI 架构经历完整重写。v7 将动画系统迁移到 TLM 的魔法咏唱控制器，解决了多女仆状态共享、跨线程数据可见性、动画死循环等关键问题。

最近完成了一轮六阶段模块化重构：

  • 消除重复 — ConditionOperator、MatchMode、applyMath 三处 100% 重复代码合并
  • 解决循环依赖 — ExpressionResolver 和 ConditionEvaluator 移入 core 包，core 包从 3 处外部依赖降到 1 处
  • 统一数据模型 — fromData() 转换逻辑从 core 层内聚到 data 层 (toCore())
  • 拆分超大文件 — MoreActionAPI 从 500 行拆为 3 个聚焦类 + 113 行委托存根
  • GUI 修复 — SelectParam 正确渲染为下拉框（非输入框）、条件运算增加数值输入、武器动画布局修复
  • 引擎超时修复 — 动画阻塞超时从固定 600 tick (30秒) 改为按动画实际时长自适应
