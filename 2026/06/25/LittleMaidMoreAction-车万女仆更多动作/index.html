项目概述

LittleMaidMoreAction 是 Touhou Little Maid (车万女仆/TLM) 的附属模组。通过可视化规则编辑器，玩家可以为女仆配置复杂的战斗动作 —— 无需编写任何代码或 JSON。

核心流程: 触发事件 → 条件匹配 (100个条件) → 执行动作序列 (75种动作)

技术架构

六边形架构 — 基于 v5 规则引擎设计：

core/ (零 Minecraft 依赖, 可独立单元测试)
  ├── spi/        类型安全参数系统 (sealed TypedParam)
  ├── registry/   注解扫描 + 三层回退 (ForgeClassScanner→ClassGraph→BuiltinRegistrar)
  ├── engine/     异步管道 (ActionPipeline) + 条件匹配器 (ConditionMatcher, params-aware)
  ├── model/      不可变数据模型 (record) + 树形条件 (AND/OR/NOT)
  └── cache/      规则索引 (O(1)) + 条件缓存 + 静态预计算
adapter/
  ├── tlm/        29 @SubscribeEvent → RuleEngine + MagicCasting Provider (INSTANT/FULL)
  └── gui/        Visitor 模式动态表单
impl/
  ├── condition/  100 @RuleCondition (注解驱动, 自动注册)
  └── action/     75 @RuleAction

v8.2 关键特性

  • 条件参数 Pipeline: ConditionDef → CondRow → ConditionEditScreen → ConditionCache → ConditionMatcher，全程贯通参数化条件
  • SelectionScreen: 全屏列表选择器，仿 Minecraft LanguageSelectScreen，替代 CycleButton 用于100条件/75动作的选择
  • 瞬时条件时间窗口: is_mainhand_attack 等瞬时条件触发后20tick内保持有效
  • P0 Bug修复: target_type 命名空间匹配、is_mainhand_attack 反转逻辑、1.20.1 API 兼容

安装与使用

  1. 安装 Touhou Little Maid ≥1.5.0 + Cloth Config
  2. 下载 LittleMaidMoreAction.jar 放入 mods/
  3. 游戏内 → Forge 模组列表 → LittleMaidMoreAction → 打开规则编辑器
  4. 自定义动画放入 config/littlemaidmoreaction/animations/ → F3+T
